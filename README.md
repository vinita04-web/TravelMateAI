# ✈️ TravelMate AI

TravelMate AI is a smart travel planning web application developed using Streamlit and Python. It helps users plan trips, estimate budgets, discover local experiences, manage group expenses, and analyze travel spending through an interactive dashboard.

---

## 📌 Features

### 🏠 Home
- Overview of TravelMate AI features.
- User-friendly interface.

### 🔐 Login & Sign Up
- Simple authentication system.
- Users can create an account and log in.
- Logout functionality included.

### 🗺️ Trip Planner
- Select a destination.
- Choose trip duration.
- Generate itinerary recommendations.

### 🍽️ Local Experiences
- Discover cultural experiences and activities.
- Explore hidden gems.
- Get restaurant recommendations based on budget.
- View famous dishes and Google Maps links.

### 💰 Budget Predictor
- Estimate trip expenses automatically.
- Calculates:
  - Hotel Cost
  - Food Cost
  - Transport Cost
- Provides budget suggestions.
- Displays recommended hotels.

### 👥 Expense Manager
- Manage group travel expenses.
- Add group members.
- Split expenses equally.
- View settlement details.

### 📊 Analytics Dashboard
- Expense summary.
- Cost per traveler.
- Expense breakdown table.
- Pie chart visualization.
- Bar chart comparison.
- Budget health indicator.
- Download expense report as CSV.

### ℹ️ Destination Insights
- Best season to visit.
- Ideal trip duration.
- Average budget.
- Top attractions.

### 📝 Trip History
- Save planned trips.
- View previous trips.
- Download trip history report.
